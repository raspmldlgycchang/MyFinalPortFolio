Image Viewer App

이미지에서 캡쳐 영역을 SET ROI버튼을 통해 조각이미지를 출력하는 프로젝트입니다

캡쳐 구현

출처 : 본인(GYCCHANG), 작업인원 :1(본인)

기능 : 최신 캡쳐 영역을 화면에 남겨두는 기능

기능 : SET ROI버튼을 통해 지정한 영역 좌표 출력

기능 : 조각이미지 출력버튼을 통해 캡쳐 영역 출력

Video

가장 최신 캡쳐 영역을 SET ROI를 통해 파일열기에서 영역 출력가능

한계 및 느낀점

한계

그림에 그려진 기존의 캡쳐 영역이, SET ROI버튼을 두 번 눌러 영역 지정을 해야만, 지워진다.
조각이미지 출력될 때 캡쳐로 지정한 영역을 컨트롤과 파일에서는 제대로 가져오는데,
조각이미지를 가져오는 코드인, 비트맵 정보를 CImage에 복사하는 함수에서,
윈도우 영역과 Client 영역의 캡션 차이로 인해서,
조각이미지로 가져오는 부분이 좌표가 그만큼씩 밀려서 복사된다

느낀점

윈도우와 클라이언트 영역 간의 캡션, 한글과 영문의 글자 차이에 따른 글자체, 글자크기 설정 등 세세한 부분이
시각성을 결정하는 것이 영상처리 분야에서 주니어 개발자 딱지를 떼기위한 첫 단계라고 생각한다.
나의 꼼꼼함과 기본적인 내용 구현 가능한 상태에서
우리 디알젬 회사에서 ERP/MRP 개발유지보수 업무를 통해
완성도 높은 영상처리 프로그램을 만들 수 있는 공부를 하고 싶다.
