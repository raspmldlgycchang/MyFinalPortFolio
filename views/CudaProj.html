Reduction Sum

토너먼트 방식으로 배열의 원소의 합을 구하는 방식입니다

기능 및 구현 방법

출처 : 학교강의자료, 작업 인원 :1(본인)

기능 : __ syncthreads()를 통한 스레드 동기화

구현 및 효과 : warp / kernel 개수 축소로 병렬처리 연산횟수 감소

느낀 점

reduction tree와 dfs알고리즘로부터 유추가능한
inverse reduction tree알고리즘을 작성하여 구현할 수 있었습니다.
이로부터 알고리즘 문제에서 2차원배열을 다루는데 자주 사용되었던
dfs알고리즘이 영상처리 분야에서도 쓰인다는것을 알았고
우리 디알젬 회사에서 기존의 알고리즘을 응용하여 더 많은 병렬처리로
효율적인 영상처리를 할 수 있는 공부를 하고 싶습니다.

Brent-Kung Inclusive Scan

prefix sum, parallel scan을 통한 partial sum

출처 : 본인(GYCCHANG), 작업인원 :1(본인)

구현 : tournament 방식으로 구한 stride를 활용한 partial_sum이용한 배열 전체 합 구하는 알고리즘

구현 : three-kernel scan에서 third kernel scan을 reduction tree와 inverse reduction tree 코드로 구현

느낀점

스스로 dfs알고리즘의 유사성을 떠올려 inverse reduction tree로도 구현에 성공했다.
원문에서 Brent-Kung scan알고리즘이 reduction tree와 유사하다는 설명 뒤에
stride에 따른 각 반복에서 더해질 원소의 인덱스에 1을 더해서 인덱스를
설정해주어야 하는 이유를 생각해보도록 했고 이에 대한 답변을
구하기 위해 reduction tree를 적용해보고
그 이후에 dfs알고리즘을 써도 되겠다는 생각을 했다. 영상처리에 있어서
기존의 알고리즘을 응용해서 더 많은 효율적인 코드를
만들어 낼 수 있겠다는 생각에 이 분야에 더 관심이 가게 되는 계기였다.

Shared Memory와 gpu을 이용한 행렬곱셈가속화 알고리즘

토너먼트 방식으로 배열의 원소의 합을 구하는 방식입니다

기능 및 구현 방법

출처 : 본인(GYCCHANG), 작업 인원 :1(본인)

기능 : __ syncthreads()를 통한 스레드 동기화

기능 : 수학의 행렬곱셈 속도보다 250,000배 빠른 속도의 연산의 병렬처리

구현 : Shared memory를 이용해 행렬 일부를 블록 단위의 sum을 진행해 행렬 곱셈을 가속화

느낀 점

reduction tree와 dfs알고리즘로부터 유추가능한
inverse reduction tree알고리즘을 작성하여 구현할 수 있었습니다.
이로부터 알고리즘 문제에서 2차원배열을 다루는데 자주 사용되었던
dfs알고리즘이 영상처리 분야에서도 쓰인다는것을 알았고
우리 디알젬 회사에서 기존의 알고리즘을 응용하여 더 많은 병렬처리로
효율적인 영상처리를 할 수 있는 공부를 하고 싶습니다.
